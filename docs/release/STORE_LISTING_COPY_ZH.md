## 完整介绍
《末日工具箱》是一款面向断网、野外、灾害和极端环境场景的离线生存工具应用。

核心能力包括：
- 本地 AI 生存助手：模型下载后可在设备端离线推理，支持文字与图片输入，输出简短、可执行的生存建议。
- 离线生存知识库：内置生火、找水、搭建庇护所、急救等基础知识，无网络也能查看。
- SOS 求救模块：支持手电筒闪烁、屏幕闪烁和摩尔斯电码求救。
- 生存工具：提供指南针和 GPS 数值显示，帮助用户在无地图场景下判断方向和位置。

产品特点：
- 核心功能离线可用
- 建议简短直接，强调可执行性
- 无账号、无社交、无复杂操作
- 适合应急准备、户外备用和断网场景使用
